# 示例文件说明

本目录包含从模板项目中提取的示例文件，供参考使用。

## 目录结构

### 前端界面示例 (示例/前端界面示例/)
展示如何创建Vue前端界面组件。

### 流式楼层界面示例 (示例/流式楼层界面示例/)
展示如何创建流式楼层界面。

### 脚本示例 (示例/脚本示例/)
展示各种脚本功能的实现方式。

### 角色卡示例 (示例/角色卡示例/)
完整的角色卡示例，包含：
- 角色信息配置
- 世界书设置
- 界面组件
- 脚本逻辑

## 使用方法

这些示例文件可以作为参考，帮助你：
1. 了解项目结构
2. 学习如何组织代码
3. 复制和修改以创建自己的功能

## 注意

- 示例文件仅供参考，不会被构建到最终的HTML文件中
- 如需使用示例代码，请复制到 `src/` 目录下对应的位置
